Help

To use this editor, select a enviroment, click on a tile and click on the position it must be on game. Key bindings(ALT+):

ALT+r:
Fix the position of the last inserted tile

ALT+h:
Show this modal again

ALT+x:
Deletes the last clicked tile

ALT+l:
Load or reload stage json

ALT+y:
Generate stage from seed

ALT+n:
Generate generic floor

ALT+g:
Generate final JSON file to include on the game

ALT+c:
Clean the cursor, to stop to insert elements every click

ALT+8:
Moves the last clicked tile UP

ALT+2:
Moves the last clicked tile DOWN

ALT+4:
Moves the last clicked tile LEFT

ALT+6:
Moves the last clicked tile RIGHT
